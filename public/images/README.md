# Images

Replace these placeholder files with real photographs. Keep the same file names (or update references in `/data/*.js`).

## Folder structure
- `hero/` – hero / banner images (recommended 1920×1080, webp/jpg)
- `services/` – service illustrations (recommended 800×600)
- `doctors/` – doctor portraits (recommended 600×700, square crop works too)

## Guidelines
- Use `.webp` or optimised `.jpg` for fast loading.
- Keep file sizes below ~300 KB where possible.
- Use consistent aspect ratios for each folder.
